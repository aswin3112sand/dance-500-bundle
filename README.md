# AS DANCE Premium Bundle Platform

The definitive Tamil dance training platform. Master 639 professional steps with lifetime access and expert guidance.

## Tech Stack
- **Frontend**: React + Vite + Bootstrap 5 + GSAP (Cinematic Motion)
- **Backend**: Spring Boot 3 + Spring Security + JWT
- **Database**: MySQL / H2
- **Payments**: Razorpay Integration

## Quick Start

### 1. Backend
```bash
mvn spring-boot:run
```
Default Admin: `admin@asdance.com` / `admin123`

### 2. Frontend
```bash
cd frontend
npm install
npm run dev
```

## Features
- **639 Professional Steps**: Categorized into Easy (194), Medium (219), and Hard (226).
- **Tamil Training**: All instructions and video content in clear Tamil.
- **Glassmorphism UI**: Premium dark neon aesthetic.
- **GSAP Animations**: Smooth, cinematic reveals and scroll-triggered motion.
- **Instant Unlock**: Content becomes available immediately after Razorpay verification.

## Content Breakdown
- **Easy Steps**: Foundation techniques and basic rhythm.
- **Medium Steps**: Intermediate combinations and style variations.
- **Hard Steps**: Advanced choreography and complex sequences.
- **Real-Time Songs**: Practice with full songs in real-time.
